# Give Me Links

## What is it ?

A simple scrapper that get all links of movies on a specific website.

## How I proceed ?

1. chmod the 2 files

  ```bash
  chmod +x ./install.sh
  ```

2. Installing and getting packages

  ```bash
  ./install.sh
  ```

3. Run the package

  ```bash
  python3 gml.py
  ```
