# Give Me Links

## How I proceed ?

1. Setting the virtual env

  ```bash
  python3 -m venv env
  source ./env/bin/activate
  python -m pip install beautifulsoup4 selenium pandas
  ```

2. Download : <https://github.com/mozilla/geckodriver/releases>

3. Copy geckodriver in /usr/local/bin

4. Run the package

  ```bash
  python3 -m gml
  ```
